# Khatwaat Adaptive AI Assessment Backend

This is the Python FastAPI backend for the Khatwaat Adaptive AI Assessment platform, hosted on Hugging Face Spaces.

## Architecture
- **FastAPI**: Production-grade Python ASGI web framework.
- **Docker**: Customized slim Python container optimized for scientific libraries.
- **IRT & CAT Engines**: Probabilistic Item Response Theory (2PL/3PL) ability estimation and Computerized Adaptive Testing.
- **AI Recommendations**: Nezam El Masarat High School Track classifier and University College placement.
